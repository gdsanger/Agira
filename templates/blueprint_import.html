What is Blueprint Import?

Import allows you to load blueprints that were exported from this or another Agira instance.

Expected Format

The JSON file must contain:

  • schema_version: Version identifier (currently "1.0")
  • blueprint: Blueprint data including title, description, category, etc.
Category Handling

If the category from the imported blueprint doesn't exist, it will be created automatically.

Update vs. Create

By default, import always creates a new blueprint. Enable "Update if exists" to update an existing blueprint with the same title and category instead.

Example Export
{
  "schema_version": "1.0",
  "blueprint": {
    "title": "Example",
    "description_md": "...",
    "category": {
      "name": "Features",
      "slug": "features"
    },
    ...
  }
}
