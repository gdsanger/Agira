How to create a GitHub Personal Access Token:
  1. Go to GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token" → "Generate new token (classic)"
  3. Give it a descriptive name (e.g., "Agira Issue Creation")
  4. Select the following scopes: repo (Full control of private repositories)
  5. Click "Generate token" and copy the token immediately (you won't be able to see it again)
  6. Paste the token in the field above
